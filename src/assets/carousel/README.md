# 🎠 Carousel Images Folder

## ✨ Automatic Image Discovery

**The carousel now automatically discovers ALL images in this folder!**

Simply add your image files here and they will appear in the carousel automatically - no configuration needed!

## 📁 How to Add Images

### Step 1: Add Your Image
Place your image file in this folder (`src/assets/carousel/`)

### Step 2: Refresh
That's it! The carousel will automatically pick up the new image on refresh.

**No configuration file updates needed!**

## 📋 Image Requirements

- **Formats Supported**: `.jpg`, `.jpeg`, `.png`, `.webp`
- **Recommended Resolution**: 1920x1080 or higher (16:9 aspect ratio)
- **File Size**: Keep under 500KB for best performance
- **Naming**: Name files however you like - they'll all be included!

## 💡 Tips

- The carousel automatically sorts images alphabetically by filename
- Use descriptive filenames for better alt text (e.g., `construction-site-singapore.jpg`)
- Images are optimized by Vite during build
- All images in this folder will be included - no need to configure anything
